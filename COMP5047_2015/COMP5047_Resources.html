Arduino Assignment (due week 4)

The first assignment is designed to ensure that everyone in the class has a basic knowledge of the Arduino hardware, programming language and development environment.

The Arduino kits contain a light sensor and LEDs. Your task is to develop code to run on the arduino that will monitor the light level and turn on an LED when the light level gets below a specified value.

To do this assignment you will need to:

  • download and install the Arduino development environment on your laptop
  • setup the hardware so the arduino can monitor the light level using the light sensor and control an LED
  • write an arduino program to do the task
  • download the program to the arduino and test it

You then need to demonstrate your program to the tutor and show them the source code, then email the source code to “bob.kummerfeld+comp5047@gmail.com”.
